Static site generators such as pelican [insert link] and Jekyll [? insert link] allow you to write website content in plain text formats like Markdown and quickly convert that content into webpages. You can do the same with org-mode! All without leaving the safety of your Emacs application.

This guide is based off work by Bernt Hansen and his awesome org-mode post here, with help from this org-mode tutorial. The org-mode manual is also very helpful on this topic and is well worth a read.

This post won't be an introduction to Emacs [insert link] or Spacemacs, and since I'll be using Spacemacs, instructions might be a little bit different if you're using Emacs.

This website is published using org-mode and can be found here. My dotfiles (including my .spacemacs file) can be found here.

Getting started

To get started, let's go ahead and create a website folder. 

my-website/
   |- css/                     ;; We'll keep css files here 
   |  `- stylesheet.css        ;; Website stylesheet
   |- img/                     ;; We'll keep image files here
   `- index.org                ;; Our website homepage
